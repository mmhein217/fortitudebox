# Fortitude Learning Box Website

This is the static website for Fortitude Learning Box, built with HTML, CSS, and JavaScript.

## How to Run Locally

You can run this website in two ways:

### Option 1: Open File Directly
1.  Navigate to the `d:\fortitudebox` folder in your file explorer.
2.  Double-click on `index.html`.
3.  The website will open in your default web browser.

### Option 2: Use a Local Server (Recommended)
If you have Python installed, you can run a simple HTTP server:

1.  Open a terminal (PowerShell or Command Prompt).
2.  Navigate to the project directory:
    ```powershell
    cd d:\fortitudebox
    ```
3.  Run the server command:
    ```powershell
    python -m http.server 8000
    ```
4.  Open your browser and go to `http://localhost:8000`.

## Project Structure

-   `index.html`: The main homepage.
-   `styles.css`: Stylesheet for design.
-   `script.js`: JavaScript for interactivity.
-   `assets/images/`: Images used in the site.
-   `assets/documents/`: Downloadable reports.

## Troubleshooting

-   **Images not loading?** Ensure the `assets` folder exists and contains the images.
-   **Changes not showing?** Refresh your browser (Ctrl+R or F5).
